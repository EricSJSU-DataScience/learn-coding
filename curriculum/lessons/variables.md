## Concept

A **variable** is a name that stores a value, created with `=`. Python has several basic data types:

- `str` — text, e.g. `"Ada"`
- `int` — whole numbers, e.g. `13`
- `float` — decimal numbers, e.g. `5.2`

You can print several values at once by separating them with commas inside `print()`.

## Example

```python
name = "Ada"        # a string
age = 13             # an int
height = 5.2         # a float
print(name, age, height)
```

## Notes

- Variable names can use letters, numbers, and underscores, but can't start with a number.
- `=` means "store this value," not "equals" like in math.
- `print(a, b, c)` prints all three separated by spaces.
