## Concept

Python programs are made of instructions the computer runs one at a time, top to bottom. The most basic instruction is `print()` — it displays text on the screen.

Text in Python is called a **string**, and it always goes inside quotes — either `"double quotes"` or `'single quotes'`.

Anything after a `#` is a **comment**. Python ignores comments completely — they're just notes for humans.

## Example

```python
print("Hello, world!")   # this line prints text to the screen
# Comments start with a hash and are ignored by Python
```

## Notes

- `print()` shows whatever is inside the parentheses.
- Strings need matching quotes on both ends.
- Use comments to leave yourself notes.
