## Description

Given `grades = {"Ada": 92, "Sam": 81, "Lee": 74}`, print Sam's grade.

## Starter

```python
grades = {"Ada": 92, "Sam": 81, "Lee": 74}

# Print Sam's grade
```

## Hint

print(grades["Sam"])

## Check

```python
out = _stdout.strip()
if out == "81":
    result = "PASS"
else:
    result = f"Expected 81 but got '{out}'"
```
