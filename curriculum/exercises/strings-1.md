---

## Description

Given `message = "we did it"`, print the message in ALL CAPS.

## Starter

```python
message = "we did it"

# Print the message in ALL CAPS
```

## Hint

Use message.upper()

## Check

```python
out = _stdout.strip()
if out == "WE DID IT":
    result = "PASS"
else:
    result = f"Expected 'WE DID IT' but got '{out}'"
```
